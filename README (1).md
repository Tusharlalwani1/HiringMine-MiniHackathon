# Hiring Mine - Landing Page Clone
### Mini Hackathon Project 🏆

A pixel-perfect clone of the [HiringMine](https://www.hiringmine.com/miners) landing page built with **HTML**, **CSS**, **Bootstrap 5**, and deployed on **Vercel**.

---

## 🌐 Live Demo

👉 [hiring-mine-mini-hackathon.vercel.app](https://hiring-mine-mini-hackathon.vercel.app)

## ✨ Features

- ✅ Fully responsive (Mobile, Tablet, Desktop)
- ✅ Navbar with logo and hamburger menu on mobile
- ✅ Hero section with #BeTheShiners card
- ✅ Founder & Co-Founder profile cards
- ✅ Meet the Team section
- ✅ Culture at Hiring Mine photo grid
- ✅ Hover animations on team members and culture images
- ✅ Footer with social media icons
- ✅ Deployed on Vercel

## 🚀 Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/your-username/hiring-mine-mini-hackathon.git
```

### 2. Open the project
```bash
cd hiring-mine-mini-hackathon
```

### 3. Run it
Just open `index.html` in your browser — no setup needed! 🎉

| Device | Screen Size | Layout |
|--------|------------|--------|

| 💻 Tablet | 768px - 992px | Adjusted padding |

| 📱 Small Mobile | < 576px | Compact layout |
